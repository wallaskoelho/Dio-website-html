Desvendando as Tags HTML: Elementos Essenciais para Construir Páginas Web

aventurar no universo da criação de páginas web começa com o entendimento das tags HTML, os elementos fundamentais que estruturam e organizam o conteúdo em um documento. Vamos explorar mais detalhadamente algumas das tags HTML mais essenciais e entender como cada uma delas contribui para moldar a experiência de navegação.

<html> A Estrutura Base

A tag <html> é a espinha dorsal de qualquer página web. Envolvendo todo o conteúdo da página, ela sinaliza ao navegador que o documento está escrito em HTML, a linguagem padrão para a construção de páginas web.

Exemplo de código:

  

<!DOCTYPE html>

 <html>

  <head>

   <title>Minha Página</title>

  </head>

  <body>

   <h1>Olá, mundo!</h1>

  </body>

</html>

Resultado:

  
    
    
    
      Minha Página
    
    
      
Olá, mundo!

<head>: Metadados e Configurações

Dentro do <html>, encontramos a tag <head>, onde residem informações cruciais sobre o documento. Aqui, definimos o título da página, links para estilos (CSS), scripts e outros metadados essenciais para a renderização e interação do usuário.

Exemplo de código:

  

<!DOCTYPE html>

 <html>

  <head>

   <title>Minha Página</title>

   <link rel="stylesheet" href="estilos.css">

   <!-- Outros metadados -->

  </head>

  <body>

   <h1>Olá, mundo!</h1>

  </body>

</html>

<body>: O Corpo da Página

A tag <body> abraça todo o conteúdo visível da página. Textos, imagens, links e outros elementos visuais residem aqui, criando a experiência única que os visitantes verão e interagirão.

Exemplo de código:

  

<!DOCTYPE html>

 <html>

  <head>

   <title>Minha Página</title>

   <link rel="stylesheet" href="estilos.css">

   <!-- Outros metadados -->

  </head>

  <body>

   <h1>Olá, mundo!</h1>

   <h2>Bem-vindo à Minha Página</h2>

   <!-- Outros elementos -->

  </body>

</html>

Resultado:

  
    
    
      
        Minha Página
      
    
      
Olá, mundo!

Títulos e Subtítulos 
<h1>, <h2>,---, <h6>

Essas tags são utilizadas para definir a hierarquia de títulos e subtítulos na página. O <h1> é o título mais importante, enquanto <h6> é o menos importante.

Exemplo de código:

  

<!DOCTYPE html>

 <html>

  <head>

   <title>Minha Página</title>

   <link rel="stylesheet" href="estilos.css">

   <!-- Outros metadados -->

  </head>

  <body>

   <h1>Titulo h1</h1>

   <h2>titulo h2</h2>

   <h3>Titulo h3</h3>

   <h4>titulo h4</h4>

   <h5>Titulo h5</h5>

   <h6>titulo h6</h6>

   <!-- Outros elementos -->

  </body>

</html>

<a>: Links

<ul> e <li> : Listas Não Ordenadas

As tags <ul> e <li> são utilizadas para criar listas não ordenadas. <ul> envolve toda a lista, enquanto <li> define cada item da lista.

Exemplo de código:

    

  <!DOCTYPE html>

  <html>

    <head>

    <title>Minha Página</title>

    <link rel="stylesheet" href="estilos.css">

    <!-- Outros metadados -->

    </head>

    <body>

    <h1>Titulo h1</h1>

    <h2>titulo h2</h2>

    <p>Este é um parágrafo de exemplo.</p>

    <a href="https://dio.me" target="_blank">Visite nosso site</a>
    
    <ul>
      <li>Item 1</li>
      <li>Item 2</li>
      <li><a href="https://www.dio.me/" target="_blank">DIO.me</a></li>
      <li><a href="https://www.google.com/" target="_blank">Google</a></li>
    </ul>

    <!-- Outros elementos -->

    </body>

  </html>

<ol> e <li> : Listas Ordenadas

Da mesma forma que as listas não ordenadas, as tags <ol> e <li> são fundamentais para estruturar informações em listas, mas agora de maneira ordenada. <ol> envolve a lista, enquanto <li> define cada item, indicando a ordem numérica.

Exemplo de código:

        
    
      <!DOCTYPE html>
    
      <html>
    
        <head>
    
        <title>Minha Página</title>
    
        <link rel="stylesheet" href="estilos.css">
    
        <!-- Outros metadados -->
    
        </head>
    
        <body>
    
        <h1>Titulo h1</h1>
    
        <h2>titulo h2</h2>
    
        <p>Este é um parágrafo de exemplo.</p>
    
        <a href="https://dio.me" target="_blank">Visite nosso site</a>
        
        <ul>
          <li>Item 1</li>
          <li>Item 2</li>
          <li><a href="https://www.dio.me/" target="_blank">DIO.me</a></li>
          <li><a href="https://www.google.com/" target="_blank">Google</a></li>
        </ul>

        <ol>
          <li>Item 1</li>
          <li>Item 2</li>
          <li><a href="https://www.dio.me/" target="_blank">DIO.me</a></li>
          <li><a href="https://www.google.com/" target="_blank">Google</a></li>
        </ol>
  
    
        <!-- Outros elementos -->
    
        </body>
    
      </html>

Ao utilizar <ol>, você proporciona uma organização sequencial, útil para instruções passo a passo, rankings, ou qualquer conteúdo que exija uma ordem específica.
